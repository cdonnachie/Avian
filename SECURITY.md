# Security Policy

## Supported Versions

The latest release of Avian Core is supported with security updates.
See https://avn.network for current releases.

## Reporting a Vulnerability

To report security issues, please open a private security advisory on the
[Avian GitHub repository](https://github.com/AvianNetwork/Avian/security/advisories/new)
or contact the development team through our community channels at https://avn.network.

Please do not file public issues for security vulnerabilities.
